# ESP32 Automated Dog Feeder

An IoT-enabled dog feeder that dispenses food on a schedule or on-demand 
via a WiFi-connected web interface. Built with an ESP32 microcontroller 
and servo motor actuation.

---

## Features

- WiFi-connected control interface accessible from any device on the network
- On-demand and scheduled feeding modes
- Servo-driven dispensing mechanism
- Lightweight web UI hosted directly on the ESP32
